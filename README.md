## Setup
### Local backend/frontend server (we will host these 2 ltr)

	cd backend
	npm run dev
	
	cd frontend
	npm run dev

- run both servers at once or just backend only (2 separate terminals)
- backend is for the api endpoints (u can use postman to test them)
- frontend is on port 5173 (http://localhost:5173/)
